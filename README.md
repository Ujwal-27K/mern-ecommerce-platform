# MERN E-commerce Platform

This is a full-stack MERN (MongoDB, Express, React, Node.js) e-commerce platform.

## Setup

1. Backend: `cd backend` && `npm install` && `npm start`
2. Frontend: `cd frontend` && `npm install` && `npm start`

## Features

- User authentication
- Product management
- Shopping cart
- Order processing
